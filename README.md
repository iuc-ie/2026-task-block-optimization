# TULAŞ Görev Blokları Optimizasyonu

## Girdi Dosyası
VeriSeti.xlsx

## Çıktı Dosyası
VeriSeti_Ciktisi.xlsx

## Proje Amacı
Bu proje, TULAŞ görev bloklarını optimize etmek amacıyla hat-sefer atamaları gerçekleştirmektedir. Sistem, Excel formatındaki verileri analiz ederek uygun görev bloklarını belirler ve optimize edilmiş atama sonuçlarını yeni bir Excel dosyası olarak üretir.

## Kullanılan Teknolojiler
- Python
- Pandas
- OpenPyXL

## Çalıştırma

```bash
python main.py
```
